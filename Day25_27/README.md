**目标**

学习 JavaScript 的函数，学习如何使用和封装函数，学习Date对象

**任务**

1、请写一个GoSteps函数，然后跑通测试用例

2、我们现在来做一个最简单的时钟，通过小练习来学习 Date，复习定时，然后再练习一下函数的封装

具体需求如下：

a.在页面中显示当前日期及时间，按秒更新

b.格式为 YYYY 年 MM 月 DD 日 星期 D HH:mm:ss

c.注意位数的补齐，比如：

-- 假设时间为2008年10月10日星期一的12点12分12秒，显示2008年10月10日星期一 12:12:12

-- 假设时间为2008年1月1日星期一的3点2分2秒，显示2008年01月01日星期一 03:02:02

d.编码过程中，注意对函数的封装，尽量让一个函数就做一个事情，而不要把所有的功能、代码揉在一起

3、完成上面需求后，现在需求做一些小的变更

a.输出格式变为：2008-10-10 Monday 07:10:30 PM

b.基于上面的需求，要求，同时在页面上，输出两种格式的日期时间

4、编写JavaScript代码

a.当变更任何一个select选择时，更新result-wrapper 的内容

b.当所选时间早于现在时间时，文案为 现在距离 "所选时间" 已经过去 xxxx

c.当所选时间晚于现在时间时，文案为 现在距离 "所选时间" 还有 xxxx

d.注意当前时间经过所选时间时候的文案变化

e.注意选择不同月份的时候，日期的可选范围不一样，比如1月可以选31天，11月只有30天，注意闰年
